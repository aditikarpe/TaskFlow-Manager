# TaskFlow-Manager

A simple console-based task management system built with Python, SQLite, and Flask.  
This project allows you to add, view, update, and delete tasks with priorities and statuses.  
It also runs a Flask server in the background using multithreading, showcasing concurrent execution.

---

##  Features
- Add new tasks with description and priority (High/Medium/Low).
- View all tasks with status (Pending/Complete).
- Update task status.
- Delete tasks by ID.
- Persistent storage using SQLite database.
- Flask server running in the background for demonstration.

---

##  Tech Stack
- Python (Core programming language)
- SQLite (Database)
- Flask (Web framework)
- Threading (For running Flask and console menu together)
